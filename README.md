# Atwood Knives Notifier
Checks Atwood Knives to see if the bike tool has been released.

## To use:
- copy file
    - `cp secrets.py.example secrets.py`
- Fill in your info
- Install requirements (use a virtualenv)
    - `pip install -r ./requirements.txt`
